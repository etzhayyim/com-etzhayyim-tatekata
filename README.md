# tatekata 建方

Construction robotics orchestration actor for civil and MEP work within the
declared low-rise scope. This is the standalone west repository
`etzhayyim/com-etzhayyim-tatekata`.

EDN is canonical for the actor manifest, cell configuration, lexicons, schema,
and kotoba seed state. Implementations live under `src/tatekata/`, tests under
`test/tatekata/`, and generation tooling under `tools/`.

```bash
bb test
bb -m tatekata.kotoba.ingest-mcp --dry-run
```

The constitutional gates preserve open firmware, IPFS-pinned site evidence,
multi-robot witness quorum, deterministic trajectories, safety and sourcing
audits, Murakumo-only inference, member-held keys, and consent-bound outward
actions.
